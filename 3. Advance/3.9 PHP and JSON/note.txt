PHP and JSON

-> What is JSON ?
* JSON stands for Javascript Object Notation.
* Its a syntax for storing and exchangin data.
* Since JSON format is a text-based format, it can easily be sent to and from a server, and used as a data 
format by any programming language.

-> PHP and JSON
* PHP has some built-in functions to handle JSON.
1. json_encode()
2. json_decode()
