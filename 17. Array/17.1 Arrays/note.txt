-> What is Array?
= An array is a special variable that can hold many rules under a single name, and you can access
the values by referring to an index number or name.

-> Type of Array
1. Indexed = Arrays with numeric index
2. Associative = Arrays with named keys
3. Multidimensional = Array contain more than one array
