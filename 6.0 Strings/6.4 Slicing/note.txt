Slicing String

-> This function allows to return a range of character by using the substr().
-> To use: specify the start index and the number of characters you want to return.

Slicing to the end
-> By leaving out the length parameter, the range will go to the end

Slicing from the end
-> Use the negative indexes to start the slice from the end of string
->  The last character has index -1.

Negative length
-> use negative length to specify how many character to omit, starting from the end
of string.
