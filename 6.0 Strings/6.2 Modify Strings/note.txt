Lesson 6.1 

Modify Strings

1. Uppercase
-> strtoupper() = functions return string in uppercase

2. Lowercase
-> strtoupper() = function return string in lowercase

3. Replace string
-> str_replace() = function replaces some character with some other character in a string

4. Reverse string
-> strrev() = function to reverse string

5. Remove whitespace
-> trim() = remove any whitespace from the beginning or the end.

6. Convert string to array
-> explode = functions that will split strings into an array.
-> the first parameter of the explode() function represent the "separator". 
The separator specifies where to split the string
