PHP File Handling

-> File handling is an important part of any web application. You often need to open and process
a file for a different task.

PHP manipulating files function:
1. Create (C)
2. Read (R)
3. Upload (U)
4. Edit (E)

Be Careful when manipulate files
common error:
1. edit the wrong file.
2. filling hard-drive with garbage data.
3. deleting the content of a file by accident.
