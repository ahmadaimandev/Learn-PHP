PHP - Math
-> PHP has a set of math functions that allow you to perform a mathematical task on numbers.


1. pi() function
-> Return value of PI.

Code syntax:
echo (pi());

2. min() and max() functions
-> min() and max() functions can be used to find the lowest and highest value in a list of arguments.

Code syntax:
echo(min(0, 150, 30, 20, -8, -200));
echo(max(0, 150, 30, 20, -8, -200));

3. abs() function
-> abs() function returns the absolute (positive) value of a numbers.

Code syntax:
echo(abs(-100));

4. sqrt() function
-> sqrt() function return the square root of number 

Code syntax:
echo(sqrt(100));
