$_POST

-> $_POST contain an array of variables received via the HTTPS POST method.
-> there are two main ways to send variables via the HTTP post method:
1. HTML forms
2. Javascript http request

$_POST in html forms    
-> A HTML form submits information via the HTTP POST method if the form's 
attribute is set to "POST"

$_POST in Javascript HTTP request
-> When sending a HTTP request in Javascript, you can specify that the 
HTTP method is post
